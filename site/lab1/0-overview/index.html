    实验概述

    1. 实验目的

      （1）进一步熟悉RARS、Logisim等汇编和模拟仿真工具的使用；

      （2）熟练掌握RISC-V汇编语言，熟悉并理解RISC-V指令系统；

      （3）了解程序在单周期RISC-V CPU搭建的SoC中的运行过程。

    2. 实验内容 (for miniRV)

    题目1：基本输入输出

      在提供的Logisim环境中，运行提供的程序Exercise1.asm，并阅读该程序源码，分析程序功能，学习汇编程序如何访问I/O接口及外设。

      数码管、LED、拨码开关的端口地址如表1-1所示。

    表1-1 数码管、LED、拨码开关端口地址

    外设 基址
    数码管 0xFFFF_F000
    LED 0xFFFF_F060
    拨码开关 0xFFFF_F070

      根据实验步骤运行程序，熟悉Logisim工具及操作流程。

    题目2：简易计算器设计

      运用miniRV指令集编写汇编程序，实现简易计算器。计算器是输入、输出都是8bit的数据。

      计算器执行逻辑运算时，需将输入的操作数A和B视为 无符号数；执行其他运算时，需将输入的操作数A和B视为 有符号数 。

    注意

      计算器程序最终需要下板运行和演示，因此 禁止使用除miniRV指令集以外的其他指令。

      如果后续设计CPU时，只实现24条必做指令，则编写本题目时不要使用附加的13条指令。

      计算器的运算符和操作数A、B均通过拨码开关输入，如表1-2所示。

    表1-2 8位简易计算器的拨码开关输入定义

    拨码开关 SW[23:21] SW[20:16] SW[15:8] SW[7:0]
    输入功能 运算符 保留 操作数A 操作数B

      计算器的运算结果需显示到数码管DK7~DK0。具体地，计算器的功能如表1-3所示。

    表1-3 运算器功能定义

    SW[23:21] 运算类型 数码管显示内容
    3'b000 A & B 运算结果以2进制显示
    3'b001 A | B 运算结果以2进制显示
    3'b010 A ^ B 运算结果以2进制显示
    3'b011 A << B 运算结果以有符号整数显示
    3'b100 A >> B (算术右移) 运算结果以有符号整数显示
    3'b101 (A == 0) ? B : [B]补 运算结果以2进制显示
    3'b110 A B 除法的 商 以有符号整数显示

    提示

      除法可使用《计算机组成原理》中的恢复余数法或加减交替法等算法来实现。
      PPT回顾：

      以上两种算法也适用于整数除法，注意试商是从被除数最高位开始的。符号位单独处理，取除数和被除数的绝对值进行运算。
      设被除数 = +1101B，除数 = -0110B；的绝对值 = 01101B，的绝对值 = 00110B，的补码 = 11010B，则有：

      令被除数减去，即加上：
      · 若得到的余数为负数，表明不够减，故商上0，并加上以恢复余数，然后被除数左移1位，再加上；
      · 若得到的余数为正数，表明够减，故商上1，令被除数左移1位，再加上；
      重复以上步骤N次，N为位宽。在本例中，N = 4。

      令被除数减去，即加上：
      · 若得到的余数为负数，表明不够减，故商上0，令被除数左移1位，再加上；
      · 若得到的余数为正数，表明够减，故商上1，令被除数左移1位，再加上；
      重复以上步骤N次，N为位宽。在本例中，N = 4。

    2A. 实验内容 (for miniLA)

    题目1：基本输入输出

      在提供的LoongArchAssembler汇编及模拟工具中，运行提供的程序Exercise1A.asm，并阅读该程序源码，分析程序功能，学习汇编程序如何访问I/O接口及外设。

      数码管、LED、拨码开关的端口地址同表1-1。

      阅读LoongArchAssembler简介文档，熟悉汇编及模拟工具的操作流程。

    题目2：简易计算器设计

      题目内容与miniRV的类似，但有以下不同：

      （1）只能使用miniLA指令集中的指令实现计算器；如果后续设计CPU时，只实现25条必做指令，则编写本题目时不要使用附加的13条指令；
